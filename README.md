# Electron AutoTheme

The specified function is triggered at sunset and sunrise.

## Example

```ts
import { AutoTheme } from "electron-autotheme";

function setWindowTheme(useDark: boolean) {
    /* ... */
}

new AutoTheme(setWindowTheme);
```
